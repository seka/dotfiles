以下の手順でデイリースタンドアップ用サマリーを生成してください。

1. `git log --since="24 hours ago" --oneline --all` で直近24時間のコミットを取得する
2. `git status` でコミットされていない変更（進行中の作業）を確認する
3. コミットメッセージとブランチ情報から作業内容を日本語でまとめる

## 出力フォーマット

```
## デイリースタンドアップ（YYYY-MM-DD）

✅ **完了した作業**
- （コミット内容から箇条書き）

🔄 **進行中**
- （コミットなしの変更がある場合のみ記載）

❓ **ブロッカー**
- （該当する場合のみ記載。なければこのセクションは省略）
```

## ルール
- コミットがない場合は「昨日はコミットなし」と明示する
- 複数ブランチにまたがる場合はブランチ名を付記する
- 技術的な詳細より「何を達成したか」を重視する
- 日本語で出力する
